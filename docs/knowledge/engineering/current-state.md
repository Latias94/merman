U1 capability tracking is now implemented as `docs/lsp/CAPABILITIES.md` plus a parser-backed family matrix test in `merman-lsp`; `FenceTextIndex::from_text` now refuses to project payload-only directive lines such as `click`, `linkStyle`, `accTitle`, `accDescr`, and `title` into node IDs or outline entries;
