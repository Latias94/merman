# AAD-090 — Route Coverage And Sequence Mirror Contract

Date: 2026-06-02

## Summary

- Fixed a reported `flowchart TD` branch/merge regression where a right/down branch bend rendered
  as a disconnected Unicode `└` instead of a connected `┐`.
- Routed TD same-rank merge edges through the existing LR direct route planner so `C --> D` in a
  branch merge is rendered instead of silently dropped.
- Kept Mermaid-compatible sequence output aligned with Mermaid 11.15 `mirrorActors=false`, then
  added explicit bottom participant boxes through `AsciiRenderOptions::with_sequence_mirror_actors`
  and `merman-cli --sequence-mirror-actors`.
- Fixed namespace-qualified class relation endpoints in core so namespace members are reused instead
  of duplicated as fully qualified top-level classes.
- Fixed shared class/ER spanning relation lanes so cross-level edges route around intermediate boxes
  instead of writing through entity/class text.
- Added route-plan and parser-backed tests so these issues are caught by automation instead of
  manual inspection.
