## Notes

The current graph renderer intentionally supports only the tracer-bullet subset: boxed nodes,
direct left-to-right edges, direct top-down edges, and linear node order. Subgraphs, non-adjacent
routing, multi-root layout, edge labels, arrow variants, and richer Mermaid shape semantics remain
for ARP-050+.
