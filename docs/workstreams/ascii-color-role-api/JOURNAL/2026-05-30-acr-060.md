# ACR-060 Journal

Date: 2026-05-30

## Scope

Implemented the flowchart Mermaid foreground style-mapping subset for `classDef`, `class`, inline
`style`, and `linkStyle`.

## Notes

- Added parser-backed red tests for node class/inline style colors, class statements on nodes and
  subgraphs, and linkStyle edge/label colors.
- Extended the internal `Canvas` color slot from role-only storage to role-or-direct RGB storage.
- Added `graph/style.rs` to resolve typed flowchart style arrays into graph node, group, and edge
  foreground styles.
- Preserved default plain output by applying colors only during ANSI/HTML finalization.
- Kept `fill`, background, stroke width, and other CSS properties as documented no-ops rather than
  mapping them to misleading foreground output.
