# ACEG-030 Shared Relationship Layout Boundary

Date: 2026-05-30
Owner: codex

## Summary

Introduced `crates/merman-ascii/src/relation_graph.rs` as the minimal shared terminal placement
boundary for class and ER relationship rendering. The module owns rendered box identity, stacked box
output, vertical center calculation, centered text/marker lines, and final vertical stack assembly.

Class and ER adapters now route existing single-relationship outputs through this shared seam while
retaining all diagram-specific semantics locally:

- Class keeps relation type, marker side, line style, labels, and unsupported diagnostics in
  `class/render.rs`.
- ER keeps cardinality, identifying/non-identifying line style, labels, and unsupported diagnostics
  in `er/render.rs`.
