# ACEG-050 ER Multi-Relationship Rendering

Date: 2026-05-30
Owner: codex

## Summary

Implemented the ER multi-relationship layout slice on top of the shared `relation_graph` box seam.
The ER adapter now supports layered DAG layouts where every relationship connects adjacent levels
and can be drawn without crossing:

- simple relationship chains, preserving cardinality and labels between each entity pair,
- simple relationship stars, preserving each label and leaf cardinality marker,
- explicit diagnostics for crossing, cyclic, parallel, spanning-level, and unrelated-entity layouts.

ER relationship semantics remain in `er/render.rs`: `cardA`/`cardB`, identifying versus
non-identifying line style, and role labels are still typed-model-driven.
