# ACEG-040 Class Multi-Relationship Rendering

Date: 2026-05-30
Owner: codex

## Summary

Implemented the first classDiagram multi-relationship layout slice on top of the shared
`relation_graph` box seam. The class adapter now supports layered DAG layouts where every relation
connects adjacent levels and can be drawn without crossing:

- simple extension stars, such as one parent with multiple children,
- simple extension chains, such as grandparent to parent to child,
- explicit diagnostics for crossing, cyclic, parallel, spanning-level, and unrelated-class layouts.

The shared seam gained read-only box geometry and canvas drawing helpers. Class-specific semantics
remain in `class/render.rs`: marker orientation, relation line style, labels, and unsupported
diagnostics are not pushed into the shared module.
