## Summary

Added `relation_components` in `relation_graph` and routed class/ER rendering through component
adapters. The helper validates endpoints, keeps all relationship-bearing boxes in one layered
planner domain, and emits isolated boxes as standalone components joined by a blank line.

This policy preserves the adjacent-layer crossing behavior added in
`ascii-class-er-topology-routing`; splitting by strict graph connectivity would incorrectly render
two disjoint relationships as two separate diagrams.
