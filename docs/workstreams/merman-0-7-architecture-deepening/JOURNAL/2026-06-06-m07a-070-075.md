# M07A-070 / M07A-075 Journal

Date: 2026-06-06
Status: Done

## Summary

- M07A-070 introduced shared root viewport planning in the SVG parity layer and migrated only
  `treeView` as the proof family.
- `treeView` now supplies content bounds; root `viewBox`, fixed/responsive width/height, and root
  style emission are produced through `RootViewportPlan`.
- Parser and render-side `treeView` depth guards were kept to protect both Mermaid source parsing
  and host-supplied typed render models.
- M07A-075 closed the PDF size-limit gap found during headless boundary review. PDF conversion now
  validates intrinsic SVG dimensions before vector conversion and before CLI PDF page wrapping.

## Remaining Boundary Findings

- Raw external SVG input still needs a dedicated trusted/untrusted boundary review.
- `foreignObject` fallback transform parsing remains string-based.
- Config numeric domain constraints remain family-local.
- Gantt time control is available internally but not exposed through all headless entry points.
