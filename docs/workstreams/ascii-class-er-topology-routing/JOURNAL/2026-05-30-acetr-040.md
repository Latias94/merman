# ACETR-040 Closeout

## Summary

Closed the ASCII class/ER topology routing lane after shipping adjacent-layer crossing support by
layer reordering.

Public support docs now describe classDiagram and erDiagram crossing support as bounded to layouts
that can be resolved by reordering a lower layer by parent order.

## Review

The shared planner still owns only terminal structure. Class and ER adapters keep typed
relationship semantics, markers/cardinalities, labels, and diagnostics.

Remaining topology work is intentionally split:

- dense label/marker collision routing,
- parallel relationship routing,
- cyclic or spanning-level routing,
- unrelated component placement.
