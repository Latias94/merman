# ACETR-030 Shared Crossing Planner

## Summary

Implemented adjacent-layer crossing support in the shared planner by reordering each non-root layer
by previous-layer parent order before crossing validation and placement.

This turns the two-by-two `A -> D`, `B -> C` shape into a readable layout where the lower layer is
ordered `D`, `C`. The planner still validates structural crossings after reordering, so denser
topologies that cannot be made readable remain diagnosable.
