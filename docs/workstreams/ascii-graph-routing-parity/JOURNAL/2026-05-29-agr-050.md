# AGR-050 Journal

Date: 2026-05-29

## Change

Hardened the simple subgraph layout slice:

- Group titles now render as centered rows inside the group box instead of being embedded into the
  top border.
- Non-empty group layouts reserve title space above member nodes and bottom padding below them.
- Empty subgraphs no longer force ordinary graph layouts to shift right/down.
- `FLOWCHART_SUPPORT.md` now describes grid layout support and the remaining subgraph boundary.
- The graph fixture allowlist increased from 37 to 44 exact matches.
