# AGR-030 Journal

Date: 2026-05-29

## Change

Replaced the LR-only linear layout assumption with a reference-style grid placement slice:

- LR nodes are placed on 3x3 logical grid cells, with roots stacked by component and children placed
  one level to the right.
- Column and row widths are derived from node labels, edge length modifiers, and configured graph
  padding.
- Same-row, same-column downward, down-then-right, and right-then-up edge routes are drawn from the
  grid-derived node positions.
- Widened-column node labels now use upstream-compatible centering.
- The graph fixture allowlist increased from 13 to 31 exact matches.
