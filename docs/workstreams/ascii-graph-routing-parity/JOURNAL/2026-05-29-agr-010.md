# AGR-010 Journal

Date: 2026-05-29

## Change

Split `crates/merman-ascii/src/graph/mod.rs` into graph-native model types and a
flowchart adapter boundary:

- `graph/model.rs` contains `AsciiGraph`, node/edge/group types, shape enums, and graph builders.
- `graph/adapter.rs` contains `FlowchartV2Model` validation and conversion.
- `graph/mod.rs` keeps the renderer/layout code and imports only graph-native types.

This keeps `merman-core` dependencies out of the renderer, which is the boundary needed before
larger routing parity work.
