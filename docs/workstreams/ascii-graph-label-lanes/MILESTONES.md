# ASCII Graph Label Lanes - Milestones

Status: Complete
Last updated: 2026-05-29

## M0 - Scope And Evidence Freeze

Exit criteria:

- Workstream docs exist.
- First executable task is bounded.

## M1 - Routing Module Deepening

Exit criteria:

- `routing.rs` delegates path planning, drawing, and labels to private modules.
- Current fixture parity remains unchanged.

## M2 - Label Lane Parity

Exit criteria:

- Duplicate or bidirectional label fixtures move to the exact allowlist, or the blocker is split
  with concrete evidence.
- Existing exact fixtures remain green.

## M3 - Closeout

Exit criteria:

- Broad gates pass.
- Docs and gap inventory reflect final parity.
- Lane is committed or explicitly handed off.
