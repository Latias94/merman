## Summary

Added parser-backed coverage for long flowchart subgraph titles, implemented automatic wrapping in
the shared graph-label/layout/draw path, updated support docs, and closed the lane.

## Changes

- Added `flowchart_parser_long_subgraph_title_wraps_to_multiple_rows`.
- Added `GraphLabel::wrapped` so explicit hard breaks and automatic wrapping share one label model.
- Changed group layout to wrap titles against the existing group content width instead of expanding
  the group to the raw title width.
- Updated group drawing to consume the precomputed `GraphLabel`.
- Updated flowchart support docs and README summaries.
