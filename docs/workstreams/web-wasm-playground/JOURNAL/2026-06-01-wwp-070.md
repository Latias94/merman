# WWP-070 Mermaid Compare Mode

Added an optional playground compare mode for browser-to-browser inspection against Mermaid JS.

Changes:

- Added `mermaid@11.15.0` to the playground dependency set.
- Added `playground/src/lib/mermaid-renderer.ts` with a lazy dynamic import and normalized render
  result.
- Extracted pan/zoom SVG rendering into `SvgViewport`.
- Added a `Compare` preview tab with side-by-side Merman and Mermaid JS panes.
- Added per-pane copy SVG, export SVG, and export PNG actions.

Validation:

```bash
npm run build --prefix playground
```

Browser smoke:

- Loaded `http://127.0.0.1:5173/`.
- Confirmed Mermaid JS was not loaded on the default SVG preview.
- Opened `Compare`.
- Confirmed Merman and Mermaid JS rendered two SVG panes.
- Confirmed Mermaid JS loaded only after opening `Compare`.
- Screenshot: `target/playground-compare-smoke.png`.

Result: passed. The build still reports the existing large chunk warning.
