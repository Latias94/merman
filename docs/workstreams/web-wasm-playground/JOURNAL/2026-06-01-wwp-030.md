# WWP-030 - TypeScript Web Package

Date: 2026-06-01
Status: DONE

## Summary

Added `platforms/web` as the `@merman/web` TypeScript package. It builds the Rust WASM crate with
`wasm-pack`, compiles TypeScript declarations and JavaScript into `dist`, and verifies package
contents before pack/publish.

## Implementation Notes

- `build:wasm` writes generated wasm-bindgen files to `platforms/web/pkg`.
- `scripts/clean-pkg.mjs` removes wasm-pack's generated `pkg/.gitignore` so npm packaging can see
  the generated files.
- `scripts/prepack-check.mjs` fails pack/publish when `dist` or generated WASM files are missing.
- `src/index.ts` provides typed options and an initialization cache around `pkg/merman_wasm.js`.
- Generated `dist`, `pkg`, and local `node_modules` are ignored.

## Evidence

```bash
npm install --prefix platforms/web
npm run build --prefix platforms/web
npm run prepack --prefix platforms/web
cd platforms/web && npm pack --dry-run
cargo fmt --check
git diff --check
```

All gates passed. `npm pack --dry-run` listed `dist`, generated JS, and the `.wasm` binary in the
tarball. NPM publication remains deferred.
