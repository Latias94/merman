# 2026-05-28 - AIF-020 through AIF-040

## Summary

Implemented the indexed FCoSE boundary and wired Architecture layout to it.

## Changes

- Added `manatee::algo::fcose::IndexedGraph`, `IndexedNode`, `IndexedCompound`, `IndexedEdge`,
  `IndexedFcoseOptions`, indexed constraints, and `layout_indexed`.
- Kept the existing `layout(Graph, FcoseOptions)` API as a compatibility adapter.
- Converted FCoSE spectral compound bookkeeping from string-keyed maps to indexed parent/order data.
- Replaced Architecture's transient string-keyed `manatee::Graph` construction with direct indexed
  FCoSE input.
- Added a manatee regression test proving indexed and compatibility FCoSE outputs match for a
  compound/constraint graph.
