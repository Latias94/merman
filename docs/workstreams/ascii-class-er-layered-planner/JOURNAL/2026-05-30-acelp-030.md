# ACELP-030 ER Adapter Extraction

## Summary

Routed erDiagram layered relationship rendering through the shared
`relation_graph::plan_layered_relation_boxes` planner.

The ER adapter now adapts each relationship into a generic top/bottom edge with label metadata for
placement. It still owns cardinality marker direction, identifying versus non-identifying line
style, relationship labels, and user-facing diagnostic wording.
