# ACELP-040 Closeout

## Summary

Closed the ASCII class/ER layered planner lane. Both classDiagram and erDiagram layered relationship
rendering now use `relation_graph::plan_layered_relation_boxes` for terminal box planning.

## Review

The shared planner owns generic terminal structure only: directed box IDs, level assignment,
parallel/unrelated/cyclic/spanning/crossing structural validation, label-aware relation gaps, row
grouping, centering, and placed box coordinates.

Class and ER adapters still own typed relationship semantics, marker/cardinality drawing, labels,
line styles, and public diagnostic wording. No dense/crossing topology support was added.
