# ACELP-020 Class Adapter Extraction

## Summary

Added a shared terminal-layout-only layered planner in `relation_graph` and routed classDiagram
layered relationship rendering through it.

The shared planner owns:

- generic directed edges over `RelationGraphBox` IDs,
- source/sink level assignment,
- missing endpoint, parallel edge, unrelated box, cyclic, spanning-level, and crossing structural
  categories,
- row grouping, global centering, label-aware vertical gaps, and placed box coordinates.

The class adapter still owns class-specific relationship semantics and maps generic structural
errors back to the existing `AsciiError::UnsupportedFeature` text.
