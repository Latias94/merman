# Changelog

## 0.1.0

- Initial preview release of the Merman VS Code extension.
- Adds local Mermaid diagnostics, completion, hover, symbols, references, rename, code actions, semantic tokens, preview, SVG export, PNG export, SVG copy, and PNG copy.
- Packages `merman-lsp` and `merman-cli` runtime binaries per platform through `bin/<platform>-<arch>/`.
